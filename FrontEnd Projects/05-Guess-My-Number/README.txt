=====================
Guess My Number
=====================

🔢 A simple number guessing game.

Functionality:
--------------
- User guesses a number between 1-20
- Tracks score and high score
- Feedback on each guess

Code Structure:
---------------
- All logic in a single JS file
- Inline styling manipulation via DOM

Concepts Practiced:
--------------------
✔ DOM selection and manipulation  
✔ Conditional logic  
✔ Game logic with functions

How to Run:
-----------
Open `index.html` in browser.
install live server
 run command 'live-server'
