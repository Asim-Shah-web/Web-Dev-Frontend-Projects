=====================
Pig Game
=====================

🎲 A 2-player dice game built with vanilla JavaScript.

Functionality:
--------------
- Roll dice and accumulate points
- First to 100 wins
- Switch turns and reset game

Code Structure:
---------------
- Procedural code structure
- Single HTML/CSS/JS file

Concepts Practiced:
--------------------
✔ DOM manipulation  
✔ Game state management  
✔ Event-driven programming

How to Run:
-----------
Just open `index.html` in your browser.
 install live server
 run command 'live-server'
