=====================
Drum Kit Game
=====================

🥁 A browser-based virtual drum kit.

Functionality:
--------------
- Press keyboard keys to play drum sounds  
- UI animation on key press

Code Structure:
---------------
- JS event listeners  
- Sound triggering via Audio()

Concepts Practiced:
--------------------
✔ Key event listeners  
✔ DOM manipulation  
✔ Audio API basics

How to Run:
-----------
Open `index.html`.
install live server
 run command 'live-server'
