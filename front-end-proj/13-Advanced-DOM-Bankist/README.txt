=====================
Bankist App - Home Page
=====================

🎨 A landing page for the Bankist app showcasing modern UI design.

Functionality:
--------------
- Smooth scroll navigation  
- Sticky navbar  
- Tabbed content and section reveal  
- Lazy loading images

Code Structure:
---------------
- Clean modular code inside script.js  
- All animations and transitions using JS/CSS

Concepts Practiced:
--------------------
✔ Intersection Observer API  
✔ DOM traversal and manipulation  
✔ UI animations  
✔ Modular and clean code

How to Run:
-----------
Open `index.html` directly.
install live server
 run command 'live-server'
